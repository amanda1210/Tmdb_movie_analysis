Introduction¶

The dataset that I selected is TMDB movies data set. In this dataset, we have the movie data records from 1966 to 2015. There are 10866 records and 21 columns in the datasets include: release_year, genre, revenue, budget, etc.

Questions¶

  • Which genres are most popular from year to year?
  • What kinds of properties are associated with movies that have high revenues?

#Since we could have multiple types to genre for on movie, I need to seperate them to see the counts of movie for each genre
# Find unique genres
def unique(col):
    entries= list(col)
    collect=[]
    for entry in entries:
        for i in entry.split("|"):
            if i not in collect:
                collect.append(i)
    return(collect)

# count occurrence for each genre:
from collections import Counter

for y in year:
    y_new_df=new_df[new_df['release_year']== y]
    
    genres_i1=list(y_new_df['genres'])
    genres_f1= []
    
    for genre in genres_i1:
        for i in genre.split("|"):
            genres_f1.append(i)
    
    new_list= Counter(genres_f1)
    for genre in unique(new_df['genres']):
        
        if genre not in genres_f1:
            occ_df[genre].append(0)
        else:
            occ_df[genre].append(new_list[genre])
